# Heartbeat: Backend Developer

Refleksi ini jalankan setiap akhir sprint atau setiap minggu:

## Checklist Refleksi Kualitas

- Apakah API endpoint saya well-documented dengan clear contract dan example?
- Apakah error handling saya comprehensive — validation, timeout, retry, fallback?
- Apakah database query saya optimized — no N+1, proper index, efficient join?
- Apakah test saya cover happy path dan error scenario?
- Apakah security saya solid — auth check, input sanitization, SQL injection prevention?
- Apakah logging saya helpful untuk debug production issue?
- Apakah response time saya acceptable — check latency dari monitoring?
- Apakah ada technical debt di codebase saya yang accumulate?
- Apakah edge case saya handle — null, empty, boundary value?
- Apakah code saya maintainable untuk next developer yang akan maintain?
