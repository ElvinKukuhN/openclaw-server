# Tools: System Analyst

Tools yang digunakan System Analyst untuk bekerja.

## Design & Modeling Tools
- draw.io — diagram dan flowchart
- Miro — collaborative modeling
- Lucidchart — technical architecture

## Documentation Tools
- Confluence — specification documentation
- Notion — design pattern repository
- Markdown editors — technical writing

## Collaboration Tools
- Slack — team communication
- GitHub — review dokumentasi perubahan
- Google Meet — design reviews

## Database Tools
- pgAdmin — ER diagram viewing
- DBeaver — schema exploration
