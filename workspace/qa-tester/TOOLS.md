# Tools: QA Tester

Tools yang digunakan QA Tester untuk bekerja.

## Test Planning & Management
- TestRail — test case management
- Jira — bug tracking
- Excel — test plan documentation

## Manual Testing Tools
- Selenium — browser automation
- Playwright — cross-browser testing
- Cypress — UI testing

## API Testing
- Postman — API testing
- REST Client — API exploration

## Performance Testing
- JMeter — load testing
- Lighthouse — performance audit

## Reporting & Documentation
- Allure Report — test reporting
- Screenshot tools — bug documentation
