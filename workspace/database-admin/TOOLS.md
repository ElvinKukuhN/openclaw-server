# Tools: Database Administrator

Tools yang digunakan Database Administrator untuk bekerja.

## Database Management Tools
- pgAdmin — PostgreSQL management
- MySQL Workbench — MySQL management
- MongoDB Compass — MongoDB exploration
- DBeaver — multi-database explorer

## Monitoring & Maintenance
- Datadog — database monitoring
- Prometheus — metrics collection
- pg_stat_statements — query analysis

## Backup & Recovery
- pg_dump — PostgreSQL backup
- mysqldump — MySQL backup
- Veeam — backup software

## Development Tools
- SQL IDE — query development
- Database Designer — schema design
- Git — version control untuk schema changes
