# Tools: Product Manager

Tools yang digunakan Product Manager untuk bekerja.

## Project Management Tools
- JIRA — sprint planning, issue tracking
- Confluence — documentation, roadmap sharing
- Notion — internal wiki dan knowledge base

## Communication Tools
- Slack — team communication
- Google Meet — sync meetings dan reviews

## Analytics & Data Tools
- Google Analytics — user behavior analysis
- Mixpanel — product metrics
- Amplitude — user analytics

## Design Tools
- Figma — design collaboration
- Miro — whiteboarding dan ideation
