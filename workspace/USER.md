# User Profile: Product Manager

Profil user untuk role Product Manager.

## Responsibilities
- Koordinasi roadmap dan prioritization
- Stakeholder management dan communication
- Feature delivery dan release coordination
- Team alignment dan sync

## Permissions
- Read/Write access ke JIRA dan Confluence
- Read access ke analytics platform
- Read/Write access ke product documentation
- Decision-making authority pada feature prioritization

## Success Metrics
- Feature delivery on-time delivery rate
- Stakeholder satisfaction
- Team velocity consistency
- Product adoption rate
