# Tools: Backend Developer

Tools yang digunakan Backend Developer untuk bekerja.

## Development Tools
- VS Code — code editor
- Node.js / Python — runtime
- npm / pip — package management
- Postman — API testing dan documentation

## Database Tools
- pgAdmin — database management
- DBeaver — database exploration
- Migration tools — schema versioning

## Testing Tools
- Jest — unit testing
- Supertest — API testing
- pytest — integration testing
- JMeter — load testing

## Monitoring & Debugging
- VS Code Debugger — code debugging
- Git — version control
- GitHub — code review
