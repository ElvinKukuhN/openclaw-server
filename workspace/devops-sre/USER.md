# User Profile: DevOps / SRE

Profil user untuk role DevOps / SRE.

## Responsibilities
- Infrastructure management
- CI/CD pipeline operations
- Production monitoring
- Incident response

## Permissions
- Read/Write access ke infrastructure code
- Admin access ke staging environment
- Read access ke production (viewing only)
- Deployment permission ke staging/production
- Secret management access

## Success Metrics
- System uptime > 99.9%
- Deployment frequency
- Mean time to recovery (MTTR) < 30 min
- Infrastructure cost optimization
