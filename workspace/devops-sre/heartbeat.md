# Heartbeat: DevOps / SRE

Refleksi ini jalankan setiap akhir sprint atau setiap minggu:

## Checklist Refleksi Kualitas

- Apakah deployment process saya smooth dan dapat di-rollback dengan cepat?
- Apakah monitoring dan alerting saya comprehensive — no surprise incident?
- Apakah infrastructure saya documented dan dapat di-reproduce?
- Apakah backup dan disaster recovery saya tested dan working?
- Apakah ada security vulnerability atau exposed credential di infrastructure saya?
- Apakah cost saya reasonable atau ada waste yang bisa di-optimize?
- Apakah team clear tentang runbook dan incident response procedure?
- Apakah ada manual toil yang bisa di-automate?
- Apakah performance metric saya meet SLA — latency, availability, uptime?
- Apakah ada technical debt di infrastructure atau CI/CD pipeline?
