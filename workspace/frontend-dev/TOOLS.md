# Tools: Frontend Developer

Tools yang digunakan Frontend Developer untuk bekerja.

## Development Tools
- VS Code — code editor
- Node.js — runtime environment
- npm — package management
- TypeScript — type checking

## UI/UX Tools
- Figma — design file viewing dan collaboration
- Storybook — component documentation
- Chrome DevTools — debugging dan performance

## Testing Tools
- Jest — unit testing
- React Testing Library — component testing
- Cypress — end-to-end testing
- Lighthouse — performance audit

## Version Control
- Git — code version control
- GitHub — code review dan collaboration
