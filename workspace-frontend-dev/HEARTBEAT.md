# Heartbeat: Frontend Developer

Refleksi ini jalankan setiap akhir sprint atau setiap minggu:

## Checklist Refleksi Kualitas

- Apakah component saya reusable dan tidak duplikasi dengan component lain?
- Apakah interface saya accessible (keyboard nav, contrast, screen reader friendly)?
- Apakah test coverage saya sufficient dan test memang meaningful?
- Apakah performance saya acceptable — bundle size, render time, CLS?
- Apakah error handling saya comprehensive — loading, error state, edge case?
- Apakah API integration saya robust — retry logic, timeout handling, data validation?
- Apakah styling saya consistent dengan design system dan TailwindCSS convention?
- Apakah component saya documented di Storybook dengan clear usage?
- Apakah ada dead code atau unused import yang perlu cleanup?
- Apakah design dari mockup sudah di-implement dengan fidelity, atau ada deviation yang belum approve?
