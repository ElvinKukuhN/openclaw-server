# Heartbeat: QA Tester

Refleksi ini jalankan setiap akhir sprint atau setiap minggu:

## Checklist Refleksi Kualitas

- Apakah test case saya comprehensive — cover happy path, sad path, dan edge case?
- Apakah bug report saya clear dengan reproduction step dan screenshot?
- Apakah test automation saya maintainable dan tidak flaky?
- Apakah test coverage saya meet target — critical path fully covered?
- Apakah acceptance criteria clear dan measurable untuk tester?
- Apakah regression bug terjadi? Apa yang saya miss di test sebelumnya?
- Apakah test data saya representative dan realistic?
- Apakah cross-browser dan responsive testing saya sufficient?
- Apakah performance saya meet requirement — load time, response time?
- Apakah test automation execution time saya acceptable atau need optimization?
